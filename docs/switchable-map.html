Switchable Map Style

Add select menu

<div id="style-type">
    <select name="styleType" id="style-type-select">
        <option value="light">Light</option>
        <option value="dark">Dark</option>
    </select>
</div>

Add select menu style

#style-type {
    box-sizing: border-box;
    margin: 0;
    padding: 0;
    position: absolute;
    top: '16px';
    left: '16px';
    z-index: 100;
}

Assign the first select option value as the default style type. Then use event listener to set the style type of the map.

bkoigl.accessToken = '<Barikoi API Key>' // required
const map = new bkoigl.Map({
    container: 'map',
    center: [ 90.3938010872331, 23.821600277500405 ],
    zoom: 12,
    styleType: 'light'  // Pre-defined Map styleType
})

map.on('load', () => {
    // Set Style Type
    document.getElementById('style-type-select').onchange = e => {
        map.setStyleType(e.target.value)
    }
})
